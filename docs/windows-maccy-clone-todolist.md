## 功能范围基线

当前 Maccy 实际覆盖的核心范围包括：

- 剪贴板监听与历史记录
- 多格式内容保存：文本、HTML、RTF、图片、文件
- 多条 clipboard item 合并为单条历史记录
- 去重与内容合并
- 搜索：精确、模糊、正则、混合
- 固定项与快捷键选择
- 历史项元数据：来源应用、首次复制时间、最近复制时间、复制次数
- 预览面板
- 自动粘贴与无格式粘贴
- 忽略规则：按应用、按类型、按正则
- 托盘图标、快捷键唤醒、双击修饰键唤醒、快捷键循环选择   
- OCR 图片文本识别
- 开机启动、自动更新、设置持久化、设置迁移与重置

## 技术选型建议

- 绝对最小体积首选：`C++20 + Win32 API + CMake + SQLite`
- 兼顾开发速度的备选：`C# + WPF + Win32 interop`
- 当前不选：`WinUI 3`，因为 Windows App SDK 部署体积和依赖不利于“首版尽量小”
- 不建议首版使用 Electron，系统集成成本和资源占用都不划算
